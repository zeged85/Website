SPA Q&A

Answer the following questions:
1.Is your page an SPA?

Yes. This website is running on a single page. This page does not reload at any stage, nor does it link to another page.

2.What is the main technological improvment of SPA?

JavaScript is the base to it all. And the usage of AJAX allows a way to communicate with back-end servers without doing a full page refresh.

3.What are the technical approaches of creating SPA?

- There are various techniques available that enable a browser to retain an SPA.
The usage of JavaScript Frameworks are mostly Popular for Building SPAs:
Angular
React
Ember
Aurelia
Vue.js
Cycle.js
Backbone

There are also complimentary teachnologies allowing asynchronous calls to the server. such as:
Ajax
Websockets
SSE
